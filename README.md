# Instacart-Market-Basket-Analysis
This project analyzes customer purchasing behavior on Instacart using SQL, Python, and Machine Learning. The objective is to uncover hidden patterns in customer orders and predict product reordering behavior.  The project combines SQL-based data analysis with Machine Learning models and data visualization.

# Problem Statement
Understanding customer purchase behavior is critical for e-commerce platforms.
This project aims to:
• Analyze order patterns
• Identify frequently reordered products
• Predict whether a product will be reordered

# Technologies Used
• Python
• MySQL
• Pandas
• NumPy
• Scikit-learn
• Matplotlib
• Seaborn

# SQL & Data Analysis
- Complex joins across multiple tables
- Window functions
- Aggregations & subqueries
- Customer-level and product-level insights

# ML Models Used
- Logistic Regression
- Random Forest Classifier

# Model Evaluation Metrics
• Accuracy
• Precision
• Recall
• F1 Score
• ROC-AUC Score

# Features Used
- Order Frequency
- Product Reorder Ratio
- Add-to-Cart Position
- Department & Aisle
- User Purchase History
